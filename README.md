# LibgdxTextureView

Libgdx use TextureView Rendering，修改Libgdx的渲染View为TextureView，可以跟原生的view进行透明层叠，Libgdx版本为1.9.14

博客地址: [博客地址](https://skyhand.blog.csdn.net/article/details/88540497)
